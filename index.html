Busca Binária

Busca Binária é um jogo de adivinhação simples, onde o jogador deve adivinhar um número entre 1 e 100. O jogo apresenta três níveis de dificuldade, que determinam a quantidade de tentativas (vidas) que o jogador tem para acertar o número.

Descrição do Jogo

  1. Início do jogo: O jogador escolhe entre três níveis de dificuldade:
    • Fácil: 100 tentativas.
    • Médio: 15 tentativas.
    • Difícil: 7 tentativas.
  2. Escolha do número: O computador gera um número entre 1 e 100. O jogador tenta adivinhar o número.
  3. Dicas: O jogo indica se o número é maior ou menor, e quantas tentativas restam.
  4. Condições de vitória ou derrota:
    • O jogador vence se adivinhar antes das tentativas acabarem.
    • O jogador perde se não acertar dentro das tentativas disponíveis.

Torre de Hanoi

A Torre de Hanoi é um jogo matemático onde o objetivo é mover uma pilha de discos de uma haste para outra, seguindo regras específicas.

Descrição do Jogo

  1. Objetivo: Mover todos os discos da haste de origem para a haste de destino.
  2. Regras:
    • Somente um disco pode ser movido por vez.
    • Um disco maior nunca pode ser colocado sobre um disco menor.
    • Há uma haste auxiliar que pode ser usada temporariamente.

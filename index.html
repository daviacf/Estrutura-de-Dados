Advinhação com Busca Binária

Busca Binária é um jogo de adivinhação baseado em um algoritmo eficiente, onde o jogador tenta adivinhar um número escolhido pelo computador entre 1 e 100.

O que é a Busca Binária?

A Busca Binária é um algoritmo utilizado para encontrar um elemento em uma lista ordenada, dividindo repetidamente o espaço de busca ao meio. O princípio é simples: a cada tentativa, descarta-se metade dos números com base na comparação.

É amplamente usado devido à sua eficiência, com complexidade \(O(\log n)\), o que significa que reduz rapidamente o número de possibilidades.

Como funciona o jogo de Advinhação?

O jogo aplica o conceito de Busca Binária em um cenário prático. O computador escolhe um número aleatório entre 1 e 100, e o jogador tenta adivinhá-lo. A cada tentativa, o jogo informa se o número é maior ou menor, ajudando o jogador a restringir a busca.

Regras do jogo
  • Escolha do nível: O jogador começa selecionando um nível de dificuldade que define o número de tentativas:
    • Fácil: 100 tentativas.
    • Médio: 15 tentativas.
    • Difícil: 7 tentativas.
    • Impossivel: 10 tentativas (1 a 1000).
  • Escolha do número: O computador gera um número entre 1 e 100.
  • Dicas: Após cada palpite, o jogo informa se o número é maior ou menor.
  • Condições de vitória ou derrota:
    • O jogador vence se adivinhar o número antes que as tentativas se esgotem.
    • O jogador perde se não acertar dentro do número de tentativas disponíveis.
Por que usar a Busca Binária?

A Busca Binária torna o jogo mais eficiente, pois utiliza lógica para minimizar o número de tentativas. Em vez de chutar aleatoriamente, o jogador pode seguir uma estratégia de divisão ao meio, o que é muito mais rápido e eficaz.

Torre de Hanoi

A Torre de Hanoi é um jogo matemático que utiliza uma estrutura lógica semelhante à pilha para resolver o problema de movimentação de discos entre hastes.

O que é uma pilha?

Uma pilha é uma estrutura de dados linear que segue o princípio LIFO (Last In, First Out), ou seja, o último elemento a entrar é o primeiro a sair.

As operações principais de uma pilha são:

  • Push: Adiciona um elemento no topo da pilha.
  • Pop: Remove o elemento do topo da pilha.
  • Peek: Consulta o elemento no topo sem removê-lo.

Pilhas são amplamente usadas em algoritmos, incluindo o jogo Torre de Hanoi, devido à sua capacidade de organizar e gerenciar dados de forma controlada.

O que é a Torre de Hanoi?

A Torre de Hanoi é um quebra-cabeça matemático onde o objetivo é mover uma pilha de discos de uma haste para outra, seguindo regras específicas:

  • Somente um disco pode ser movido por vez.
  • Um disco maior nunca pode ser colocado sobre um disco menor.
  • É permitido usar uma haste auxiliar temporariamente.

O desafio é resolver o problema no menor número possível de movimentos, que é dado pela fórmula 2^n-1 onde n é o número de discos.

Como a Torre de Hanoi se relaciona com pilhas?

O jogo Torre de Hanoi pode ser modelado utilizando pilhas para representar as hastes:

  • Cada haste é uma pilha que armazena os discos.
  • O movimento de um disco entre hastes equivale às operações pop (remover) de uma pilha e push (adicionar) em outra pilha.
  • A lógica do jogo respeita as regras de uma pilha, garantindo que apenas o disco no topo de uma haste possa ser movido.

Ao implementar a Torre de Hanoi em programação, usar pilhas ajuda a organizar os discos e simplificar a lógica do algoritmo.

Campo Minado

O Campo Minado é um jogo que utiliza lógica e estratégia para evitar células que contenham minas escondidas. O tabuleiro do jogo pode ser representado internamente por uma matriz.

O que é uma matriz?

Uma matriz é uma estrutura de dados bidimensional organizada em linhas e colunas. Cada posição, chamada de célula, é identificada por índices que representam sua linha e coluna. É amplamente usada para representar dados organizados, como tabelas ou tabuleiros de jogos.

O que é o Campo Minado?

O Campo Minado é um jogo onde o jogador deve revelar células de um tabuleiro sem acertar as minas escondidas. Algumas células contêm números que indicam quantas minas estão nas células vizinhas, ajudando o jogador a deduzir a localização das minas.

Como o Campo Minado se relaciona com matrizes?

O tabuleiro do Campo Minado é representado como uma matriz. Aqui estão algumas das principais relações:

  • Os índices da matriz representam as posições do tabuleiro (linha e coluna).
  • Cada célula da matriz pode armazenar:
    • Uma mina;
    • Um número indicando o total de minas nas células vizinhas;
    • Um valor vazio para células seguras.
  • O cálculo dos números em cada célula baseia-se nas células ao redor (vizinhaça 8-direcional).

Portanto, o Campo Minado é um exemplo prático do uso de matrizes na resolução de problemas e na criação de jogos interativos.
